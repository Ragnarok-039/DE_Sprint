Проект № 2.
Анализ рынка валют.

Создал ETL-процесс формирования витрин данных.

Получаю данные с сайта, записываю в json.
Далее беру json и записываю в БД, где создал три таблицы.
Следующим шагом беру данные из одной таблицы, обрабатываю через SQL запрос и кладу в другую таблицу в соответствии с
требованиями задачи.

Полностью готов ETL-процесс для анализа компаний.
Хотел сделать подобное и для курса валют, но не успел. Для валют только создал таблицу и заливаю туда данные с сайта.

Весь процесс сделан на основе airflow, записан в DAG - 'exchange_rates_and_companies'.
Данный DAG работает на сервере airflow. Запускается раз в день.
